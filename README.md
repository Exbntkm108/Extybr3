# Book Tracker

**Автор:** [Коистина Головизнина]
**Описание:** Простое GUI-приложение для учёта прочитанных книг с фильтрацией по жанру и количеству страниц.
**Технологии:** Python 3.x, tkinter, json

## Возможности

- Добавление книг с проверкой корректности ввода.
- Отображение списка книг в таблице.
- Фильтрация по жанру и количеству страниц.
- Сохранение и загрузка данных из файла JSON.
- Использование Git для контроля версий.
- Примеры использования и тесты

## Тестирование 
    Тест 1: Попробуйте добавить книгу с пустыми полями — появится ошибка.
    Тест 2: Введите в поле «Страниц» нечисловое значение — появится ошибка.
    Тест 3: Добавьте несколько книг разных жанров и используйте фильтр по жанру.
    Тест 4: Используйте фильтр по количеству страниц (например, больше 300).
    Тест 5: Сохраните данные, закройте приложение и откройте снова — список должен восстановиться.
