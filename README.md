**Автор:** [Кристина Головизника]
**Описание:** Простое GUI-приложение для генерации случайных цитат с фильтрацией по автору и теме, историей и сохранением данных в JSON.
**Технологии:** Python 3.x, tkinter, json

## Возможности

- Генерация случайной цитаты.
- История сгенерированных цитат.
- Фильтрация по автору и теме.
- Сохранение и загрузка истории из файла.
- Проверка корректности ввода.
- Использование Git для контроля версий.
